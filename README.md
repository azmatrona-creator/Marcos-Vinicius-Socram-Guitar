# 🎸 Socram Instrumentos Musicais

> Um web app responsivo, elegante e funcional voltado para a exibição, curadoria e intermediação de vendas de instrumentos musicais nacionais e importados.

---

## 📌 Sobre o Projeto

O **Socram Instrumentos Musicais** é um site institucional e catálogo virtual desenvolvido para conectar músicos (desde iniciantes a profissionais) aos seus instrumentos ideais. O projeto conta com páginas dinâmicas para catalogação de produtos e canais diretos de atendimento ao cliente via e-mail e WhatsApp.

---

## 🚀 Funcionalidades Atuais

*   **Página Inicial (Home):** Apresentação da marca, missão da empresa e links rápidos de navegação.
*   **Filtro por Categorias:** Divisão clara entre Instrumentos Nacionais (`Guitar-Nacionais.html`) e Importados (`Instrumentos-Importados.html`).
*   **Seção Sobre:** Informações institucionais detalhadas sobre a loja.
*   **Canal de Vendas Direto:** Integração com links de redirecionamento para o WhatsApp corporativo e link de e-mail automatizado (`mailto:`).
*   **Design Responsivo:** Adaptado para funcionar perfeitamente em computadores, tablets e smartphones.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento Web Standard (Front-End):

*   **HTML5:** Estrutura semântica do site (`<header>`, `<main>`, `<section>`, `<footer>`, tabelas e listas).
*   **CSS3:** Estilização, layout responsivo, efeitos de *hover* e customização de tabelas e botões.
*   **JavaScript (ES6):** Manipulação do DOM e lógica para futuras integrações dinâmicas.
